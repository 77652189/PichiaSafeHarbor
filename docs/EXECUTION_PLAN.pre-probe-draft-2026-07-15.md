# PichiaSafeHarbor 项目执行计划

## 1. 决策摘要

- 当前原始 MVP 尚未进入正式候选生成阶段，科学状态仍为 `blocked`。
- 当前最大缺口不是代码能力，而是缺少足以支持全基因组边界判断的合格证据。没有该证据时，候选窗口、默认阈值、共线性核查和 Streamlit 展示都缺少可信输入。
- 当前路线的最高价值是回答一个项目级问题：公共 RNA-seq 是否足以支持可审计的 Strain-B 坐标覆盖与剪接证据轨道。它不能单独证明安全港，也不能产生 Strain-T 精确坐标。
- 推荐继续并完成已经启动的 `SRR31989028` 单重复受控探针，然后强制停在规划门禁。不得自动下载另外两个重复，不得启动全基因组边界轨道、Slice 1 或产品界面。
- 本轮只授权“单重复受控探针完成与验收”。自本计划生效起，增量预算上限为 1.5 人日、16 机器小时、额外 15 GB 临时磁盘和不超过该 run 剩余数据量的网络下载；不授权付费云资源。

## 3. 距离原始 MVP 的主要缺口

### 科学缺口

1. Strain-B 当前公开注释存在系统性边界不确定性，不能作为正式阈值和候选窗口的唯一边界来源。
2. 尚无通过验收的全基因组证据整合边界轨道；当前 `scientific_acceptance_status` 仍为 `blocked`。
3. 当前单重复数据即使通过，也只能提供 `coordinate_coverage` 和 `splice_support`；链方向仍为 `unavailable`，生物重复一致性仍未验证。
4. 目标菌株为 Strain-T，但当前主坐标是 Strain-B `GCA_001746955.1`；`exact_target_strain_coordinates=false`，尚无 Strain-T 原生组装或可靠变异映射。
5. 尚未形成默认基因缓冲距离和最小候选窗口长度的科学依据及 ADR。
6. 尚未完成 Strain-B 到 Strain-C 的候选级共线性核查。
7. 尚未完成文献整合位点的回顾性检查，更没有湿实验安全港验证。

### 产品缺口

1. 尚未进入 Slice 1，未生成正式 `candidate_window`、候选清单或排除清单。
2. 尚未完成多格式候选产物、规则版本、证据等级和独立验收的一致性闭环。
3. 尚未完成完整 MVP 验收运行。
4. Streamlit 只读展示、导航、筛选、详情、排除记录、统计、下载和错误边界均未进入授权阶段。

因此，已完成的是参考与证据资格基础设施，不是原始 MVP 本身。

## 4. 当前路线的预期决策价值

当前单重复受控探针的决策价值为“中高”，其价值来自排除路线不确定性，而不是直接交付产品：

- 能判断公开 RNA-seq 在锁定 Strain-B 坐标上是否具有可重复、非零且可解释的四染色体覆盖。
- 能判断 unique、multi、unmapped、splice 和污染/异常覆盖是否可独立复算。
- 能暴露 engineered background、外源序列、覆盖偏差和组装端点附近的不可用区域。
- 能决定是否值得投入另外两个重复，以及是否值得设计正式证据整合边界轨道。

该探针不能回答：

- Strain-T 精确坐标或 Strain-T 特异安全性；
- 链方向；
- 三重复一致性；
- 默认候选阈值；
- 候选位点的表达能力、长期稳定性或湿实验安全性。

## 8. 成功、失败与预算停止条件

### Gate P0 成功

只有同时满足以下条件，才允许规划侧考虑 Gate P1：

1. 原始文件身份、大小和哈希全部通过。
2. 四条核染色体均有非零、可解释且可独立复算的覆盖。
3. unique、multi、unmapped 和 splice 分类完整，不把未知类别包装为成功。
4. 没有全局外源污染或广泛异常覆盖；局部工程影响已明确隔离。
5. 双运行主产物一致，独立 verifier 能复算关键统计。
6. 结论严格限制在实际能力，不把非链特异数据升级为方向证据，不生成正式候选或阈值。

成功不等于自动授权三重复，也不等于科学状态已经 `accepted`。成功后需要比较：新增两个重复是否足以改变边界轨道投资决策。

### Gate P0 失败

任一以下情况触发路线停止：

- 文件身份或完整性无法确认；
- 任一核染色体缺少可解释覆盖，或覆盖/剪接结果无法独立复算；
- 存在全局污染、广泛异常覆盖或无法隔离的工程背景影响；
- 结果依赖未声明链方向、不可复现参数或主坐标之外的坐标回写；
- 预算停止线被触发且没有新的高价值证据说明应追加预算。

失败后保持 `scientific_acceptance_status=blocked`，不再下载另外两个重复。下一次规划只比较“降级产品、转向 Strain-T 原生数据、寻找真正独立证据或停止项目”。

### 预算停止线

- 人力超过 1.5 人日；
- 机器运行超过 16 小时仍不能形成可验收产物；
- 新增临时磁盘超过 15 GB；
- 下载范围超出 `SRR31989028`；
- 需要付费资源或新增外部依赖才能继续；
- 同一数据/工具故障连续重试三次仍无新的诊断证据。

触发后应停止执行并回到项目级路线评估，不得静默扩预算。

## 9. 下一次规划门禁

Gate P0 完成后只做一次项目级选择：

1. 若探针通过且另外两个重复有望显著降低边界轨道决策风险，申请 Gate P1 三重复验证预算。
2. 若探针仅支持有限坐标覆盖，但不足以改变正式边界判断，停止 RNA-seq 扩展，比较降级产品与 Strain-T 原生数据路线。
3. 若探针失败或超预算，停止当前路线，保留现有证据基础设施，不启动 Slice 1。

在 Gate P0 正式验收前，本计划不授权任何更后阶段。
